Fetch Decode Execute

Fetch

  • The PC transfers the address of the instruction or data to be fetched into the MAR
  • The MAR places the address onto the data bus
  • A read trigger is sent across the control bus that causes main memory to load the content of the location onto the data bus
  • The instruction is passed into the MDR
  • The instruction is copied from the MDR to the CIR

Decode

  • The CPU examines the instruction in the CIR and translates it in the control unit

Execute

  • The instruction within the CIR is executed by the CPU, e.g. adding two numbers, loading a number from memory to the Accumulator, or storing the contents of the Accumulator into memory
